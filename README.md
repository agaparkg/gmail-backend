## Gmail Clone App backend

#### Run these 2 commands on your terminal inside **gmail-app-backend** folder

1.  `npm install`
2.  `npm start`

#### Files we will be working on

- all our **API endpoints** goes:

  - `routes/index.js`

- data (fake database) will be located at folder:

  - `fake-db`
